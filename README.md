# qa_python
test_add_new_book_added_book_has_no_genre - проверка на отсутствие жанра у добавленной в словарь книги без жанра ('')

test_set_book_genre_success - проверка на установку жанра книги

test_get_book_genre_success - проверка на получение жанра книги

test_get_books_with_specific_genre_returns_correct_books - проверка на получение названия книги нужного жанра: 
1.выбирает нужный жанр из одной книги подходящего жанра
2.выбирает одну книгу нужного жанра из нескольких книг
3.возвращает пустой список, если нужного жарна нет
4.выбирает две книги нужного жанра из нескольких различных книг

test_get_books_genre_with_various_collections - проверка на получение словаря с книгами и жанрами:
1.добавлена одна книга
2.добавлено несколько книг

test_get_books_for_children_returns_correct_books - проверка на получение названий книг для детей (не включает в себя жанры Ужасы и Детективы):
1.выбор из одной книги подходящего жанра (комедии)
2.выбор одной книги подходящего жанра из нескольких книг
3.выбор из нескольких книг неподходящего жанра (вернется пустой список)
4.выбор двух книг подходящего жанра из нескольких книг разных жанров

test_add_book_in_favorites_book_added - проверка на добавление одной книги в избранное

test_add_book_in_favorites_cant_add_again - проверка не невозможность добавить в избранное одну и ту же книгу повторно

test_delete_book_from_favorites_removes_book - проверка на удаление одной книги из избранного

test_get_list_of_favorites_books_one_book_added - проверка на получение списка избранных книг
